 FactoryAccessControl

Система контроля посещаемости сотрудников предприятия через пропускной контроль (КПП).

 Описание
Проект реализован на ASP.NET Core 7.0 с использованием чистой архитектуры и принципов SOLID.  
Позволяет отслеживать приход и уход сотрудников на завод, вести учет смен, а также получать статистику нарушений рабочего времени.

 Особенности
- CRUD операции для сотрудников (создание, изменение, удаление, получение списка)  
- Учет смен сотрудников с временем начала и окончания, подсчет отработанных часов  
- Подсчет нарушений рабочего графика за месяц (бонусная функция)  
- Поддержка ролей сотрудников: Менеджер, Инженер, Тестировщик свечей  
- Чистая архитектура: Domain, Application, Infrastructure, API  
- Асинхронная работа с базой данных через Entity Framework Core  
- SQL Server (или SQLite) для хранения данных  
- Swagger UI для тестирования API  
- Возврат данных в формате JSON  

 Архитектура проекта
- **API** — контроллеры, точки входа в систему  
- **Application** — сервисы, бизнес-логика, интерфейсы  
- **Domain** — модели и сущности  
- **Infrastructure** — работа с базой данных, репозитории  

 Технологии
- .NET 7  
- ASP.NET Core Web API  
- Entity Framework Core  
- SQL Server / SQLite  
- Swagger / OpenAPI  
- Асинхронное программирование  
- JSON для обмена данными
